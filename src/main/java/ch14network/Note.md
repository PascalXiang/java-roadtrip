# ch14 网络编程

## 核心概念
| 概念 | 说明 |
|------|------|
| IP | 网络中设备的唯一标识 |
| 端口 | 进程的唯一标识（0-65535） |
| TCP | 可靠连接，三次握手 |
| UDP | 无连接，速度快，可能丢包 |

## TCP 通信流程
```
Server: ServerSocket → accept() → 读写流 → close()
Client: Socket → 读写流 → close()
```

## 易错点
- 端口被占用 → `BindException`
- 服务端要先启动
- 流的关闭顺序：先开后关
