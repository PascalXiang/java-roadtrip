# ch03 运算符

## 核心概念
| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+ - * / %` | 算术 | `10 / 3 = 3` |
| `== != > < >= <=` | 关系 | 结果为 boolean |
| `&& \|\| !` | 逻辑 | 短路求值 |
| `? :` | 三元 | `a > b ? a : b` |

## 易错点
- 整数除法截断小数：`5 / 2 = 2`
- `==` 比较引用类型比较的是地址，不是内容
- `&&` 短路：左边 false 右边不执行
