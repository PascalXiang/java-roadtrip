# ch03 运算符

## 核心概念
| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+ - * / %` | 算术 | `10 / 3 = 3` |
| `== != > < >= <=` | 关系 | 结果为 boolean |
| `&& \|\| !` | 逻辑 | 短路求值 |
| `? :` | 三元 | `a > b ? a : b` |
| `++ --` | 自增自减 | `a++` vs `++a` |
| `= += -= *= /=` | 赋值 | `a += 2` 即 `a = a + 2` |

## 运算符优先级（从高到低）
```
1. ()                    括号
2. ! ++ --               一元运算符
3. * / %                 乘除取模
4. + -                   加减
5. > < >= <=             关系比较
6. == !=                 相等比较
7. &&                    逻辑与
8. ||                    逻辑或
9. ? :                   三元运算符
10. = += -= ...          赋值
```

## 易错点
- 整数除法截断小数：`5 / 2 = 2`，需要 `5.0 / 2` 或 `(double) 5 / 2`
- `==` 比较引用类型比较的是地址，不是内容（String 用 `equals()`）
- `&&` 短路：左边 false 右边不执行（可利用此特性避免空指针）
- `a++` 先用后加，`++a` 先加后用

## 关键代码
```java
// 三元运算符 — 求两个数的最大值
int max = a > b ? a : b;

// 短路求值 — 避免空指针
if (str != null && str.length() > 0) { ... }

// 自增陷阱
int a = 5;
int b = a++;  // b=5, a=6
int c = ++a;  // c=7, a=7
```
