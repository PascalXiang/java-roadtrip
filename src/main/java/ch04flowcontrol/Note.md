# ch04 流程控制

## 核心概念
```
if-else if-else    多分支判断
switch-case        匹配固定值（支持 int/String/enum）
for                知道循环次数
while              先判断后执行
do-while           先执行后判断（至少执行一次）
break / continue   跳出循环 / 跳过本次
```

## 易错点
- switch 忘写 break → 穿透
- for 循环死循环条件
- do-while 末尾有分号
