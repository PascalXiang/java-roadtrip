# ch08 面向对象进阶

## 核心概念
| 概念 | 关键字 | 说明 |
|------|--------|------|
| 继承 | `extends` | 子类复用父类属性和方法 |
| 多态 | 父类引用指向子类对象 | 编译看左边，运行看右边 |
| 抽象类 | `abstract` | 不能实例化，可有抽象方法 |
| 接口 | `interface` | 纯抽象，多实现 |
| 静态 | `static` | 属于类，不属于对象 |
| 不可变 | `final` | 变量不可改 / 类不可继承 / 方法不可重写 |

## 三大特征
**继承** → 代码复用
**多态** → 灵活扩展

## 易错点
- 子类构造方法默认调用 `super()`
- `@Override` 注解可以检查重写是否正确
- 接口中的方法默认 `public abstract`
