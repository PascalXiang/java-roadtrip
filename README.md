# ☕ Java Roadtrip

> Java 零基础学习笔记 + 代码（黑马程序员课程）

## 📁 项目结构

```
java-roadtrip/
├── 01-basics/          # 基础语法
│   ├── src/            # 源码
│   └── README.md       # 笔记
├── 02-oop/             # 面向对象
├── 03-collections/     # 集合框架
├── 04-io/              # IO 流
├── 05-thread/          # 多线程
├── 06-network/         # 网络编程
├── 07-database/        # JDBC / 数据库
├── 08-framework/       # 框架入门
└── notes/              # 学习笔记
```

## 🚀 如何运行

每个模块都是独立的 Java 项目，进入对应目录运行：

```bash
cd 01-basics/src
javac HelloWorld.java
java HelloWorld
```
